# yt-transcripts

Organization-wide knowledge base of YouTube content from competitors and industry creators (primarily finance, financial advice, management, and marketing). Pulls metadata + transcripts, stores as structured markdown, ready for enrichment and eventual semantic retrieval.

## What this is

- **Catalog**: one `.md` file per video in `/output`, structured frontmatter + transcript body with inline timestamps.
- **Master index**: auto-generated `/output/index.csv` rolling up every file's frontmatter for quick filtering.
- **Distributed enrichment**: model-agnostic instructions under `/operations/enrich/` let anyone in the org enrich files using their preferred CLI (Claude Code, Codex, Gemini).
- **Retrieval-ready by design**: frontmatter, file paths, and transcript timestamps are structured so a future vector/RAG layer (Layer 3) can be added without re-processing or migrating files.

## Skills (8)

Each skill has model-agnostic instructions in `/operations/<name>/INSTRUCTIONS.md` and a thin Claude Code wrapper in `.claude/skills/<name>/SKILL.md`. Mechanical operations also expose a Python script in `/scripts/` for LLM-free runs.

| Skill | Purpose | Surface |
|---|---|---|
| `ingest` | Drop a channel or video URL → structured `.md` in `/output` | Skill + `scripts/ingest.py` |
| `enrich` | Apply enrichment instructions to files where `enriched: false` | Skill only (LLM is the worker) |
| `rebuild-index` | Regenerate `output/index.csv` from all frontmatter | Skill + `scripts/build_index.py` |
| `validate` | Check schema, paths, uniqueness, timestamp presence | Skill + `scripts/validate_corpus.py` |
| `status` | Dashboard: counts, enrichment %, transcript sources | Skill + `scripts/corpus_status.py` |
| `review-whisper` | Surface Whisper-transcribed files for spot-check | Skill + `scripts/list_whisper.py` |
| `re-enrich` | Flip selected files to `enriched: false` and re-run | Skill + `scripts/mark_unenriched.py` |
| `export-for-embedding` | Chunk corpus + dump JSONL ready for vector indexing | Skill + `scripts/export_chunks.py` |
